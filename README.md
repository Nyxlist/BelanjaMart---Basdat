# BelanjaMart - Basdat
Tugas/Project Basdat

## Open secara website(masih viewonly):
1. Import 'belanjamart.sql' ke phpMyAdmin
2. Letakkan folder belanjamart (bukan file sqlnya) di htdocs
3. Jalankan di http://localhost/belanjamart/

## ⚠️ Note Penting
Jika database **belanjamart** sudah ada, disarankan untuk menghapusnya terlebih dahulu sebelum melakukan import file `belanjamart.sql` untuk menghindari error.
